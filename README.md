# AI-суммаризатор текстов, реализованный в качестве проекта при обучении на 1 курсе магистратуры МТУСИ #

## Запуск приложения

1. Загрузить зависимости.

`pip install -r requirements.txt`

`python download_models.py`

`python download_nltk.py`

2. Поднять контейнеризированное приложение (соберётся, если нет образа)

`docker compose up --remove-orphans`

## Доступные эндпоинты

Для получения списка доступных методов и их описаний необходимо сделав вызов `http://<address>:8000/docs`
